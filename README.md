# MyMedic Monorepo

Welcome to the MyMedic project. This repository contains both the frontend and backend applications.

## Project Structure

- `backend/`: FastAPI application providing the MyMedic API.
- `frontend/`: Vite + React application for the user interface.

## Getting Started

### Prerequisites

- Python 3.x
- Node.js (Latest LTS recommended)

### Running the Backend

1. Navigate to the `backend` directory.
2. Install dependencies: `pip install -r requirements.txt`.
3. Start the server: `run_local.bat`.

### Running the Frontend

1. Navigate to the `frontend` directory.
2. Install dependencies: `npm install`.
3. Start the development server: `npm run dev`.
